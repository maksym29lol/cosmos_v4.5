# 🌌 Cosmos v4.5 — Космічний Провідник

Інтерактивний веб-додаток для дослідження космосу з каталогом Сонячної системи, 3D-візуалізацією, вікторинами, грою-стрілялкою та системою XP і рангів.

## 📁 Структура проєкту

```
cosmos/
├── index.html          ← Основна HTML-структура
├── style.css           ← Всі стилі та анімації
├── audio.js            ← Генеративний аудіодвигун
├── data.js             ← Дані про космічні об'єкти та місії
├── game.js             ← Гра «Метеоритна Атака» (Canvas 2D, ~600 рядків)
├── main.js             ← Логіка UI: навігація, вкладки, форум, порівняння, квіз
├── starfield.js        ← Анімований зоряний фон на Canvas
├── xp-system.js        ← Гейміфікація: XP, ранги, значки, конфеті
└── README.md
```

## 🚀 Як запустити

Просто відкрий `index.html` у будь-якому сучасному браузері — збірка не потрібна.

```bash
# Варіант 1: відкрити напряму
open index.html

# Варіант 2: локальний сервер (уникає проблем з CORS)
npx serve .
# або
python3 -m http.server 8080
```

## ✨ Можливості

- **Каталог Сонячної системи** — перегляд планет, супутників, галактик і космічних апаратів
- **Спектральна галерея** — вигляд об'єктів в оптичному, інфрачервоному, рентгенівському та інших діапазонах
- **Порівняння** — візуальне та табличне порівняння будь-яких двох об'єктів
- **Космічні місії** — хронологія активних, завершених і майбутніх місій
- **Живий форум** — залишай повідомлення про кожен об'єкт (зберігається в localStorage)
- **3D Сонячна система** — інтерактивна візуалізація на Three.js з режимом автопілота
- **☄️ Метеоритна Атака** — повноцінна аркадна гра з хвилями, комбо та бонусами
- **Система XP і рангів** — заробляй досвід, досліджуючи об'єкти; 7 рангів від Новачка до Легенди
- **Космічне Радіо** — генеративний ембієнтний аудіодвигун
- **Академія (Квіз)** — перевір знання про космос і отримай бонусний XP

## 🛠 Технології

| Файл | Технологія |
|------|-----------|
| `index.html` | HTML5 |
| `style.css` | CSS3 (змінні, анімації, grid/flex) |
| `xp-system.js` | Vanilla JS + Canvas 2D (малювання значків) |
| `audio.js` | Web Audio API |
| `starfield.js` | Canvas 2D анімація |
| `data.js` | Vanilla JS (шар даних) |
| `main.js` | Vanilla JS (логіка додатку) |
| `game.js` | Canvas 2D ігровий рушій |
| *(через CDN)* | [Three.js r128](https://threejs.org/) |

## 📦 Залежності

Лише одна зовнішня залежність, підключена через CDN:

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
```

Жодного npm, жодних інструментів збірки, жодного фреймворку.

## 🌐 Розгортання на GitHub Pages

1. Завантаж цю папку до репозиторію на GitHub
2. Перейди до **Settings → Pages**
3. Вибери джерело: гілка `main`, тека `/ (root)`
4. Сайт буде доступний за адресою `https://<твій-нікнейм>.github.io/<назва-репо>/`
